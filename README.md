# Suman Mishra — Portfolio

Personal developer portfolio built with Next.js 16 (App Router), TypeScript, and Tailwind CSS v4.

## Overview

This repository contains the source code for my freelance developer portfolio. It showcases my work as a Full-Stack Developer and AI Engineer, highlighting projects like FlowMine, ShieldKYC, and AutoSense.

### Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Deployment:** Vercel (Ready)

## Getting Started

### Prerequisites

Ensure you have Node.js (v18.17+) installed.

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/mishrasuman7/my-portfolio.git
   cd my-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure

- `src/app/`: Next.js App Router pages and layout.
- `src/components/ui/`: Reusable, atomic UI components (Navbar, Footer, ProjectCard).
- `src/components/sections/`: Major page sections (Hero, Projects, Skills, etc.).
- `src/lib/data.ts`: Centralised content data for easy updates.

## Deployment

This project is configured for seamless deployment on Vercel. 
Pushing to the `main` branch will automatically trigger a production build and deployment if connected to a Vercel project.
